# Supermarket Sales Analysis

This project performs Exploratory Data Analysis (EDA) on a supermarket sales dataset to identify customer behavior, sales patterns, and business insights.

## Skills & Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Highlights

- Performed data cleaning and preprocessing
- Analyzed customer demographics (Gender, City)
- Explored payment method preferences
- Evaluated product line performance
- Analyzed gross income and sales trends
- Studied customer ratings and purchase quantity patterns
- Created visualizations using Matplotlib and Seaborn
- Used correlation heatmaps to identify relationships between variables

## Key Insights

- Total customers analyzed: 1000
- Gender distribution was almost equal
- Identified the most preferred payment methods
- Compared product lines based on sales, ratings, and gross income
- Generated business insights from customer purchasing behavior
